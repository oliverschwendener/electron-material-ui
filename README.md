# electron-material-ui

This is a repo to quickly start the development of an electron application with [Material UI](https://mui.com/).

## Development

-   Install dependencies

    ```
    $ pnpm install
    ```

-   Run app in dev mode

    ```
    $ pnpm start
    ```

-   Lint files

    ```
    $ pnpm linkt
    ```

-   Perform typecheck

    ```
    $ pnpm typecheck
    ```

-   Build app

    ```
    $ pnpm build
    ```

-   Package app

    ```
    $ pnpm package
    ```

-   Package app & publish release

    ```
    $ pnpm package:publish
    ```
